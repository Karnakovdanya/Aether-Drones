# Aether Drones — Responsive Web Development, UI/UX Design & Interactions

A premium landing page designed and developed entirely from scratch. The main goal of this project was to master pixel-perfect responsive layout development, complex UI/UX user interactions, dynamic CSS animations, and basic JavaScript functionality.

## 🎯 Project Goals & Achievements
* **Full-Cycle UI/UX:** Personally designed and structured the entire user experience, content hierarchy, and visual style.
* **Hardcore Responsiveness (RWD):** Hand-coded custom breakpoints from **320px up to 1920px**, ensuring a flawless layout on all mobile devices, tablets, non-standard laptop screens, and 4K monitors.
* **Clean & Semantic Code:** Built using semantic HTML5 markup and pure CSS3 without relying on heavy third-party frameworks.

## 🚀 Key Features & CSS/JS Techniques Implemented

* **Infinite Logo Marquee (Ticker):** A smooth, automated left-to-right infinite scrolling wrapper showcasing partner companies.
* **3D Card Flip Effect (Y-Axis Rotation):** Interactive info cards that seamlessly flip 180 degrees on click to reveal additional details.
* **Hover Floating Animation:** Dynamic CSS keyframe animation on the main drone image, simulating a realistic flight/hovering effect coupled with a reactive fading shadow.
* **Animated Gradient Border:** A contact/order form featuring a lively, shifting gradient border that transitions dynamically using CSS.
* **Interactive Gallery:** A functional slider showcasing the drones in real-world environments and action shots.

## 🛠️ Tech Stack Used
* HTML5 (Semantic Architecture)
* CSS3 (Flexbox, Grid, Media Queries, Keyframe Animations)
* JavaScript (Basic)
* Responsive Web Design (RWD)
